Dentro del Body: p

La etiqueta p nos sirve para separar parrafos dentro de nuestra pagina web.
Esta etiqueta es opcional, es solo para dar formato al contenido.
Además, se puede dar una alineacion según se requiera, esto mediante el atributo ALIGN, el cual sera colocado dentro de la etiqueta p y entrecomillado, y tenemos 3 valores:

  • LEFT: el parrafo se justifica hacia la izquierda
  • CENTER: el parrafo se aliea en el centro
  • RIGHT: el parrafo se justifica hacia la derecha

EJEMPLO

<html>
<head>
<title>Parrafo y Alineacion</title>
</head>
<body>
<p>Este es un ejemplo de parrafo sin ningun valor en ALIGN </p>
<p ALIGN="LEFT">Este es un ejemplo de parrafo con ALIGN="LEFT" </p>
<p ALIGN="CENTER">Este es un ejemplo de parrafo con ALIGN="CENTER" </p>
<p ALIGN="RIGHT">Este es un ejemplo de parrafo con ALIGN="RIGHT" </p>
</body>
</html>
